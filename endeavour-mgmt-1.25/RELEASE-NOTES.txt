Endeavour Agile ALM is pleased to annouce  its 1.25 version release!

Endeavour Application Lifecycle Management is an Open Source solution to manage the creation of large-scale enterprise systems in an iterative and incremental development process. It features support for Use Case management, Iterations, Project Plan, Change Requests, Defect Tracking, Test Cases, Test Plans, Tasks, Document management, Project Wiki and many other process artifacts.
